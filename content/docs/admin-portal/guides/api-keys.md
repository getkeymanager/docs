# API Keys

This guide explains how to manage API keys in the Admin Portal.

## Overview

The API Keys section allows you to create, manage, and revoke API keys for integrating external applications with your portal.

## Key Features
- Generate new API keys
- View and manage existing keys
- Revoke compromised or unused keys

## How to Use
1. Click 'Generate API Key' to create a new key.
2. Copy and securely store the generated key.
3. Revoke keys that are no longer needed or have been compromised.
