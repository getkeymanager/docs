# Test License

This guide explains how to create and manage test licenses in the Admin Portal.

## Overview

The Test License section allows you to generate licenses for testing purposes, simulating real-world scenarios without affecting production data.

## Key Features
- Generate test licenses
- Set test parameters
- Manage and revoke test licenses

## How to Use
1. Click 'Generate Test License' to start.
2. Fill in the required test parameters.
3. Use the generated license for testing integrations or workflows.
4. Revoke test licenses when no longer needed.
