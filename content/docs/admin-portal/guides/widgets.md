# Widgets

This guide explains how to use and manage widgets in the Admin Portal.

## Overview

The Widgets section allows you to add, configure, and manage dashboard widgets for enhanced monitoring and quick access to key information.

## Key Features
- Add new widgets
- Configure widget settings
- Remove or rearrange widgets

## How to Use
1. Click 'Add Widget' to place a new widget on your dashboard.
2. Configure widget settings to display relevant data.
3. Remove or rearrange widgets as needed for your workflow.
