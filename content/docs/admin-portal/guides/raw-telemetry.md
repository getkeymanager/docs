# Raw Telemetry

This guide explains how to access and analyze raw telemetry data in the Admin Portal.

## Overview

The Raw Telemetry section provides direct access to unprocessed telemetry events and logs collected from your products.

## Key Features
- View raw event data
- Search and filter raw telemetry
- Export raw data for analysis

## How to Use
1. Navigate to the Raw Telemetry section to view incoming event streams.
2. Use search and filter tools to locate specific events.
3. Export raw data for offline analysis or troubleshooting.
