# Contracts

This guide explains how to manage contracts in the Admin Portal.

## Overview

The Contracts section allows you to create, edit, and manage contracts associated with customers and products.

## Key Features
- Add new contracts
- Edit contract details
- Link contracts to customers and products

## How to Use
1. Click 'Add Contract' to create a new contract.
2. Fill in contract details and link to the appropriate customer and product.
3. Edit or update contracts as needed.
