# Telemetry

This guide explains how to view and manage telemetry data in the Admin Portal.

## Overview

The Telemetry section provides insights into product usage, system events, and user activity collected from deployed software.

## Key Features
- View telemetry dashboards
- Filter and search telemetry events
- Export telemetry data

## How to Use
1. Navigate to the Telemetry section to view dashboards and reports.
2. Use filters to focus on specific products, customers, or time periods.
3. Export data for further analysis if needed.
