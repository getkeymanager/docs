# Products

This guide explains how to manage products in the Admin Portal.

## Overview

The Products section allows you to create, edit, and manage software products that require licensing.

## Key Features
- Add new products
- Edit product details
- Archive or delete products

## How to Use
1. Click 'Add Product' to create a new product.
2. Edit existing products as needed.
3. Archive or delete products that are no longer needed.
