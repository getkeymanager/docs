# Settings

This guide explains how to configure and manage system settings in the Admin Portal.

## Overview

The Settings section allows you to customize portal behavior, manage integrations, and set global preferences.

## Key Features
- Configure system preferences
- Manage integrations and API settings
- Set up email and notification options

## How to Use
1. Navigate to the Settings section to view available options.
2. Adjust preferences and save changes as needed.
3. Review integration and notification settings for your environment.
