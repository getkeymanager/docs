# Downloadables

This guide explains how to manage downloadable files in the Admin Portal.

## Overview

The Downloadables section allows you to upload, manage, and distribute files (such as installers, updates, or documentation) to your customers.

## Key Features
- Upload new files
- Manage existing downloads
- Set access permissions

## How to Use
1. Click 'Upload File' to add a new downloadable.
2. Manage file details and permissions as needed.
3. Remove outdated or unnecessary files.
