# Generate License

This guide explains how to generate new licenses in the Admin Portal.

## Overview

The Generate License section allows you to create new license keys for your products, specifying parameters such as product, customer, and validity period.

## Key Features
- Generate new license keys
- Select product and customer
- Set license parameters (expiry, features, etc.)

## How to Use
1. Click 'Generate License' to start the process.
2. Fill in the required details (product, customer, parameters).
3. Click 'Create' to generate the license key.
4. Copy or download the generated license key as needed.
