# Customers

This guide explains how to manage customers in the Admin Portal.

## Overview

The Customers section allows you to add, edit, and manage customer records associated with your products and licenses.

## Key Features
- Add new customers
- Edit customer details
- View customer license history

## How to Use
1. Click 'Add Customer' to create a new customer record.
2. Edit customer information as needed.
3. View a customer's license history for support or reporting.
