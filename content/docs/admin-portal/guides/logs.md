# Logs

This guide explains how to view and manage logs in the Admin Portal.

## Overview

The Logs section provides access to system and application logs for monitoring, troubleshooting, and auditing purposes.

## Key Features
- View system and application logs
- Search and filter log entries
- Export logs for analysis

## How to Use
1. Navigate to the Logs section to view recent entries.
2. Use search and filter tools to find specific log events.
3. Export logs for offline review or compliance.
