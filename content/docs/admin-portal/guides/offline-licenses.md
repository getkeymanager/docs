# Offline Licenses

This guide explains how to manage offline licenses in the Admin Portal.

## Overview

The Offline Licenses section allows you to issue and manage licenses for environments without internet connectivity.

## Key Features
- Create offline licenses
- Download license files
- Manage offline license activations

## How to Use
1. Click 'Create Offline License' to generate a new license.
2. Download the license file for distribution.
3. Track and manage activations for offline environments.
