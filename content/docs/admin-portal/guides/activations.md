# Activations

This guide explains how to manage license activations in the Admin Portal.

## Overview

The Activations section allows you to track and manage the activation status of issued licenses.

## Key Features
- View all activations
- Search and filter activations
- Revoke or reset activations

## How to Use
1. Use the search and filter options to find specific activations.
2. Click on an activation to view details or take action.
3. Revoke or reset activations as needed.
