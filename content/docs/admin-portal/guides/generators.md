# Generators

This guide explains how to use and manage license key generators in the Admin Portal.

## Overview

The Generators section allows you to configure and manage the algorithms or services used to generate license keys for your products.

## Key Features
- Add new generators
- Edit generator settings
- Enable or disable generators

## How to Use
1. Click 'Add Generator' to create a new generator.
2. Configure the generator settings as required.
3. Enable or disable generators as needed.
