# Webhooks

This guide explains how to configure and manage webhooks in the Admin Portal.

## Overview

The Webhooks section allows you to set up automated notifications and integrations with external systems based on portal events.

## Key Features
- Add new webhooks
- Edit webhook settings
- Test and monitor webhook deliveries

## How to Use
1. Click 'Add Webhook' to create a new webhook.
2. Configure the endpoint URL and event triggers.
3. Test the webhook and monitor delivery status.
