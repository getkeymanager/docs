# Admin Users

This guide explains how to manage admin users in the Admin Portal.

## Overview

The Admin Users section allows you to add, edit, and manage users with administrative access to the portal.

## Key Features
- Add new admin users
- Edit user roles and permissions
- Deactivate or remove users

## How to Use
1. Click 'Add Admin User' to create a new admin account.
2. Assign appropriate roles and permissions.
3. Edit or deactivate users as needed.
