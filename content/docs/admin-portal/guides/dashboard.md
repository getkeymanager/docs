# Dashboard

This guide covers all features and usage instructions for the Dashboard area of the Admin Portal.

## Overview

The Dashboard provides a summary of key metrics, recent activity, and quick access to important actions. Use it to monitor system health, recent logins, license usage, and more.

## Key Features
- System metrics and statistics
- Recent activity feed
- Quick links to common actions
- Alerts and notifications

## How to Use
1. Review the summary cards for a quick overview.
2. Check the activity feed for recent changes.
3. Use quick links to navigate to other portal areas.
