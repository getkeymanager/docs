## What Is This Page?

Manage enterprise contracts that allow controlled license generation by third parties (resellers, partners). Contracts provide quota-based license management.

---

## When to Use This Page

Use contracts when working with resellers, managing enterprise customers, or providing partners with controlled license generation capabilities.

---

## What You Can Do Here

### Contract Features

Product binding, generator enforcement, license quota, expiry dates, scoped API permissions, environment isolation.

![Contract Features Screenshot](placeholder-contracts-contract-features.png)

### Contract Management

Create, edit, delete contracts. Monitor license usage against quota, view contract history, revoke access, generate contract-specific API keys.
