## What Is This Page?

Configure automatic data purging policies for logs, telemetry, and other operational data.

---

## When to Use This Page

Use this page to manage storage costs, comply with data retention regulations, or configure auto-purge intervals.

---

## What You Can Do Here

### Retention Configuration

Configure auto-purge for logs, telemetry, job execution logs, webhook delivery logs.

![Retention Configuration Screenshot](placeholder-retention-policies-retention-configuration.png)
*Screenshot placeholder - will be added*

### Retention Options

1, 3, 6, 9, or 12 months. Select based on compliance requirements and storage considerations.

![Retention Options Screenshot](placeholder-retention-policies-retention-options.png)
*Screenshot placeholder - will be added*

### Protected Data

NEVER auto-purged: Audit trails, licenses, activations, contracts, products, customers. These are permanent records.

![Protected Data Screenshot](placeholder-retention-policies-protected-data.png)
*Screenshot placeholder - will be added*

### Management

Create, edit, delete retention policies. Monitor policy enforcement.
