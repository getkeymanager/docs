## What Is This Page?

View operational system logs for troubleshooting, monitoring, and security auditing.

---

## When to Use This Page

Use this page to troubleshoot errors, investigate security incidents, monitor API calls, or review system performance.

---

## What You Can Do Here

### Log Types

API calls (success and failure), authentication attempts, admin actions, system errors, job execution logs, performance metrics.

![Log Types Screenshot](placeholder-logs-log-types.png)
*Screenshot placeholder - will be added*

### Log Levels

Emergency, Alert, Critical, Error, Warning, Notice, Info, Debug.

![Log Levels Screenshot](placeholder-logs-log-levels.png)
*Screenshot placeholder - will be added*

### Log Features

Searchable (full-text), filterable (by level, type, actor), purgeable (auto-purged per retention settings), exportable.

![Log Features Screenshot](placeholder-logs-log-features.png)
*Screenshot placeholder - will be added*

### Important

Logs are operational and auto-purged. For permanent records, see Audit Trails (not shown in logs page).
