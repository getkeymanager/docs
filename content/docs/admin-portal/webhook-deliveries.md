## What Is This Page?

Track webhook delivery attempts and monitor integration health. Troubleshoot webhook failures.

---

## When to Use This Page

Use this page to verify webhook deliveries, debug integration issues, check response codes, or review retry attempts.

---

## What You Can Do Here

### Delivery Logs

Shows webhook name, target URL, event type, delivery status, HTTP response code, response body, retry attempts, timestamp.

View detailed error messages, check request/response payloads, monitor retry attempts.
