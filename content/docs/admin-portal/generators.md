## What Is This Page?

Define how license keys are created for each product. Generators control the format, activation limits, and validity rules for licenses.

---

## When to Use This Page

Use generators when setting up a new product, changing license key formats, adjusting activation limits, or configuring feature flags.

---

## What You Can Do Here

### Generator Types

UUID-based (standard UUID v4), Chunk-based (formatted segments like XXXX-XXXX-XXXX), or Custom function (user-defined logic).

![Generator Types Screenshot](placeholder-generators-generator-types.png)

### Configuration

Set activation limits, validity periods, feature flags (JSON), and metadata templates. Each generator is product-scoped and environment-scoped.
