## What Is This Page?

Manage email templates used for automated communications with customers and admins.

---

## When to Use This Page

Use this page to customize email content, add variable placeholders, preview templates, or manage different email types.

---

## What You Can Do Here

### Template Types

License issuance, assignment notifications, expiry warnings, contract notifications, customer welcome emails.

![Template Types Screenshot](placeholder-email-templates-template-types.png)

### Template Features

HTML & text versions, variable placeholders ({{license_key}}, {{product_name}}), preview support, version control.
