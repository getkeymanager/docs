## What Is This Page?

Manage downloadable files and updates that customers can access. Control versions, access rules, and file uploads.

---

## When to Use This Page

Use this page to upload new software versions, manage release files, configure access rules, or update changelog entries.

---

## What You Can Do Here

### Version Management

Semantic versioning (1.2.3), numeric versions, changelog entries per version, multiple assets per version.

![Version Management Screenshot](placeholder-downloadables-version-management.png)
*Screenshot placeholder - will be added*

### File Uploads

Drag & drop interface, supports ZIP, images, videos, PDFs. Files stored in S3 under product UUID folders with version-specific paths.

![File Uploads Screenshot](placeholder-downloadables-file-uploads.png)
*Screenshot placeholder - will be added*

### Access Rules

Define license-assigned dates, expiry constraints, time-based windows, feature flag requirements. Enforced online and offline.
