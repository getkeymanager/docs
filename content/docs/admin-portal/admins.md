## What Is This Page?

Manage administrative users who have access to the admin portal. Control roles, permissions, and account status.

---

## When to Use This Page

Use this page to create new admin accounts, modify admin roles, enable/disable admin access, or review admin activity.

---

## What You Can Do Here

### Admin List

Shows name, email, role, status (Active/Disabled), last login, and created date.

![Admin List Screenshot](placeholder-admins-admin-list.png)
*Screenshot placeholder - will be added*

### Admin Roles

Super Admin (full access), Admin (product & license management), Operator (day-to-day operations), Viewer (read-only).

![Admin Roles Screenshot](placeholder-admins-admin-roles.png)
*Screenshot placeholder - will be added*

### Admin Actions

Create new admins, edit details, change roles, enable/disable access (soft disable only), view activity log, force password reset.

![Admin Actions Screenshot](placeholder-admins-admin-actions.png)
*Screenshot placeholder - will be added*

### Security

Admins are never hard-deleted, only soft-disabled. All admin actions are audited.

![Security Screenshot](placeholder-admins-security.png)

2. Navigate to **Admin Users** in the main menu
