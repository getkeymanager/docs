# Core API Endpoints

This section contains the main API endpoints for license management, product management, contracts, telemetry, and more.

All endpoints in this section require API key authentication and follow the standard API conventions.

## Categories

### Access Restricted Area
- Validate API Access Restriction

### License Key Operations
- License verification and validation
- License activation and deactivation
- License assignment (manual and random)
- License key creation, updates, and deletion

### Product Management
- Create, update, and delete products
- Retrieve all products
- Product metadata management

### Contract Management
- Create, update, and delete contracts
- Retrieve all contracts
- Legacy contract operations

### Telemetry
- Retrieve telemetry
- Send telemetry data
- Retrieve telemetry data

### Generators
- Retrieve all generators
- Generate licenses

### Downloadables
- Access downloadable resources

### Metadata Management
- License key metadata operations
- Product metadata operations
