# WooCommerce Integration

The official WooCommerce integration plugin bridges your store with GetKeyManager, automating the delivery of license keys to your customers.

## ðŸš€ Key Features

- **Auto-Assignment**: License keys are automatically assigned when a WooCommerce order is marked as "Completed".
- **Product Mapping**: Map WooCommerce products (including variations) to specific GetKeyManager products.
- **Customer Portal**: Customers can view their purchase history and license keys directly on the "My Account" page.
- **Admin Management**: View and manage assigned licenses from the WordPress backend.
- **Environment Aware**: Toggle between production, staging, and development environments.

## ðŸ“¦ Installation

1. Download the plugin from the repository.

3. Activate via the WordPress Plugins menu.
4. Configure your API key in the **GetKeyManager** settings menu.
